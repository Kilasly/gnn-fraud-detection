# 🛡️ GNN-Based Financial Fraud Detection System

## 📌 Project Overview

This project is a Graph Neural Network (GNN)-based financial fraud detection system designed to detect anomalous and potentially fraudulent blockchain transactions using graph-based machine learning techniques.

The system uses the Elliptic Bitcoin Dataset and applies Graph Convolutional Networks (GCN) to analyze transaction relationships and identify suspicious financial activity.

A professional Streamlit dashboard was developed to provide:

- Fraud analytics
- Interactive graph visualization
- Connected transaction investigation
- Real-time fraud monitoring
- Transaction forensic analysis

---

# 🎯 Objectives

The primary objectives of this project are:

- Detect fraudulent blockchain transactions
- Analyze transaction relationships using graph structures
- Improve anomaly detection using GNNs
- Provide an interactive investigation platform
- Demonstrate real-world cybersecurity applications of graph learning

---

# 🧠 Why Graph Neural Networks (GNNs)?

Traditional machine learning models analyze transactions independently.

Graph Neural Networks improve fraud detection by:

- Capturing transaction relationships
- Modeling neighboring transaction behavior
- Detecting suspicious transaction clusters
- Learning graph connectivity patterns
- Identifying hidden fraud structures

This makes GNNs highly effective for blockchain fraud analysis.
